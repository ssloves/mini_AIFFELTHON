> 엣지화한다(없으면 dangling으로 격리 → 그래프 오염 0). 사람/LLM 없이 측정.

## 1. 해소율 (코퍼스 내 인용)

- 원시 인용: internal 4,814 + cross_corpus 2,011 = **6,825건**
- 청크별 중복 도착 제거 후 고유 인용: **6,124건**
- 실제 Article로 **해소(엣지화): 5,561건 → 해소율 90.8%**
- 미해소(dangling, 코퍼스 내인데 도착 조문 없음): 563건 (9.2%) — 엣지로 만들지 않고 격리

## 2. 해소 엣지의 항(項) 도달성 (precision 보조 지표)

- 인용이 '제N항'까지 지정한 해소 엣지: 2,990건
- 그중 해당 항이 도착 조에 실제 존재: **2,560건 → 85.6%** (정규식이 엉뚱한 조를 잡았다면 항도 불일치하는 경향)
